Open Project Dialog Box

This standard Windows dialog box is displayed when the File | Open menu option is selected (or when the equivalent hotlink is clicked).

The dialog is used to locate a SIBuilder v2 or v3 (.sip) or SIBuilder v1 (.sib) project file to be opened.

If a SIBuilder v1 or v2 project file is opened then a new file name must be selected when the project is saved. This is because SIBuilder v3 does not write .sib or v2 .sip files.
