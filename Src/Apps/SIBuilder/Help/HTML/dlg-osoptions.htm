Operating System Options Dialog Box

This dialog box is used to choose the operating systems that apply to a file group. The group is installed only on the operating systems supplied here. The dialog is accessed from the Select Options button of the Edit Group Properties dialog box.

The dialog box displays a heirachical list of supported operating systems platforms and products within a platform. The required products and platforms are selected by checking and clearing the appropriate check boxes. Checking or clearing the check box for a platform checks or clears all the check boxes for that platform's products. At least one operating system must be specified.

Any selected operating system that is not included in those specfied as valid for the whole installation is ignored. Note: the operating system which are supported by the installation project are set on the Target Operating Systems page.
