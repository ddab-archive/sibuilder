Edit Registry Template Macros Dialog Box

This dialog box is used to edit the lists of values associated with registry template macros. The dialog is opened when the Edit Macro Values button in the Choose Registry Template dialog box is clicked.

The macro whose values are to be edited is selected in the Select macro drop down list. The values associated with the macro are displayed in the Value list box. To add a new value enter it in the Value edit box and click the Add button. To edit an existing value either double click it or select it in the list box and click the Edit button. The value is then copied to the edit box. Change the value as required then click the Update button. Values can be deleted individually by selecting them and clicking the Delete button. All the values can be deleted using the Clear button.

Note: all changes made in this dialog box are irreversable – there is no way to cancel them.
