File Properties Dialog Box

This dialog box is displayed when the File Properties button on the Files page is clicked. It displays the following properties of the currently selected file:

  • File name – the name of the file to be installed.
  • Source file path – the path when the file is to be copied from.
  • Modification date – the date and time the file was last modified (this will be preserved when the file is installed and may be used when deciding whether the file will overwrite pre-exisiting files of the same name.)
  • Size – the size of the file in bytes.
  • Attributes – the file's attributes (A=Archive, R=Read Only, H=Hidden, S=System).
  • Version information – details of any version information embedded in the file. To view the different textual entries, click the required information description in the left hand list box to view the associated information in the right hand list box. This box is disabled if the file contains no version information.
