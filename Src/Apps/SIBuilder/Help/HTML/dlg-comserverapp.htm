New Delphi COM Server Application Dialog Box

This dialog box appears when the Delphi COM Apps button is clicked on the Programs page. It is used to supply the name of a local COM server application to be registered by the install program and un-registered by the uninstall program.

The dialog box presents a drop down list that contains details of all the 32 bit executable files in the current project. A program should be selected from this list.

The selected program should support the Delphi standard for registering out-of-process COM servers: i.e. passing the /regserver switch on the command line should register the program and then exit while passing the /unregserver switch on the command line should unregister the server and exit. SIBuilder cannot test for this.

On pressing the OK button SIBuilder will create two program entries on the Programs page: one to register the server and the other to unregister it. If you wish to edit these entries you should do so by selecting the program and editing in the usual way. Should you not want the server to be un-registered by the uninstaller then select the entry with the /unregserver switch and delete it.
