License Files

A license file takes the form of a plain ASCII text file that is created by the developer. The license is automatically included in the project and is displayed by the installer. By default, the license file is not installed onto the target computer.

A license file can be dragged from Explorer and dropped on the License File group box in the License Info page, providing the Display License check box is checked. The license file can be previewed in a sample dialog box.

To permanently install a license file on the target computer ensure that the file is also included in a group on the Files page.
