v2 File Conversion Comment:
License Dialog DLL Added to Project

SIBuilder uses a DLL – SITLicenseDlg.dll – to display any license required by the installer. SIBuilder 3 deletes the DLL once the license has been displayed, while SIBuilder 2 stores the DLL in the installation folder and only (optionally) deletes it with the uninstaller.

The current case is one where the DLL would have been deleted by the uninstaller under SIBuilder 2. It is not clear whether the current project requires the dialog DLL for its own use, but since the license file has been retained, it is possible. Therefore SIBuilder 3 has created a new group – "SIP2 CONVERTER: License DLL" – containing the license DLL. The group has been flagged for deletion by the uninstaller.

If it is known that the DLL is not used by the project then the DLL (and its group) can safely be deleted from the Files page.
