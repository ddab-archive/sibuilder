SIBuilder 3 Installation Information

SIBuilder Files

A SIBuilder installation comprises various files and sub-folders. The default installation is in

<Program Files Folder>\DelphiDabbler\SIBuilder3

where <Program Files Folder> is the default programs folder on your system (e.g. C:\Program Files).

Several of the files are essential to the correct operation of SIBuilder and should not be deleted. A full list of files is provided in the file ReadMe.html which can be found in the Docs sub-folder of your SIBuilder installation.

SIBuilder also stores its default project in the

<Application Data>\DelphiDabbler\SIBuilder3

folder, where <Application Data> is the per-user application data folder.

Registry Changes

SIBuilder adds the following new registry keys:

HKEY_CURRENT_USER\Software\DelphiDabbler\SITools\3.0\SIBuilder
HKEY_LOCAL_MACHINE\Software\DelphiDabbler\SITools\3.0\SIBuilder

Uninstalling SIBuilder

SIBuilder should be uninstalled via the Control Panel using the Add/Remove programs applet.

If SIBuilder is uninstalled it is important to use the provided uninstall program. This will remove installation files and tidy up the registry. However UnInstall.exe will remain in the root of your SIBuilder installation. This is because the uninstaller can't delete its own file whilst executing. This file can safely be deleted manually as can the SIBuilder3 folder. The DelphiDabbler folder can also be deleted if it is empty.

The uninstaller leaves the default project and the application data sub-directories on the system. They can be deleted manually.
