Program Errors

SIBuilder has detected one or more problems with the programs that have been set to run by the installer or uninstaller. These problems can be caused either by programs or their parameters. The offending items are marked with a symbol in the tree view on the Programs page. The likely causes of the problems, and solutions, are listed below.

Programs

There are two possible reasons why there is a problem with a program and both occur when the program has been set to run by the uninstaller:

  1. The program file is to be installed from a temporary group (and will therefore be deleted by the installer).
    Solutions:
    1. edit the program to make sure it is run only by the installer;
    2. install the program file into a different non-temporary group using the Files page.
  2. The project does not include an uninstaller.
    Solution: include an uninstall program in the project using the Project Details page.

Parameters

A parameter belongs to a program that is to be run by the uninstaller. The parameter references a file that will be installed from a temporary group.

Solutions:

  1. change the program to run only from the installer;
  2. install the parameter file into a different non-temporary folder (using the Files page);
  3. edit the parameter to refer to a non-temporary file.
