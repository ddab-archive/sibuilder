Parameters Error

SIBuilder has detected one or more problems with the parameters associated with the current program. The parameters causing the problem are marked with a symbol.

The associated program is to be run by the uninstaller while the parameter references a file that is included in a temporary group – which will not be available to the uninstaller.

Solutions:

  1. Change the program to run only from the installer (use the Execute on drop down list to select Installation only).
  2. Cancel this dialog and use the Files page to make sure the parameter file is not deleted by the installer.
  3. Edit the parameter to refer to a non-temporary file.
