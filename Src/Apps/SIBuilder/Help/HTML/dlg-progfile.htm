Choose Program File Dialog Box

This standard Windows dialog box appears when the Browse for file... item is selected from the Program combo box of the Edit Program Details dialog box. It is used to select a file to be run by the installer or uninstaller. Care should be taken to select only a program that will exist in the same or similar location on the target system.

Choose the program you wish to specify in the edit box and click OK. Only .exe, .bat or .com files should be chosen.
