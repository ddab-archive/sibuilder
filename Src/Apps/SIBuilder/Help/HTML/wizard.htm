The Project Wizard

To create an installation project you must provide various pieces of information. You enter the information into various pages of a wizard. These pages are reached from the main display by either selecting the required page from the page selector at the left of the screen or by using the Next and Previous buttons at the bottom right of the window. The pages or the wizard are:

  • Welcome Page
    Gives a brief overview of SIBuilder and provides a means of jumping off to some useful start-up tasks. This page may be hidden by clearing the Include welcome page check box in the Options dialog box.
  • Project Details Page
    Essential basic general information about the project is entered on this page.
  • License Info Page
    Used to define any license that is displayed by the installer.
  • Target OS Page
    Used to define the operating systems on which the installer will run.
  • Files Page
    Where the files to be installed and the folders they are to be installed into in are specified.
  • Programs Page
    Used to specify any additional programs to be run by the installer and/or uninstaller.
  • Registry Page
    Used to specify what modifications are to be made to the registry.
  • Build Installer Page
    Builds the install program using the information entered on the previous pages.

Brief instructions and common actions are optionally displayed at the top of the project wizard in the Info Pane.
