Tools Menu

The tools menu provides the following options:

Options Displays the Options dialog box where the following options can be edited:
  • The kind of compression used to compress install data.
  • The format used for build reports – HTML or plain text.
  • Whether the Welcome page is displayed when the program starts.
  • Whether the info pane is to be displayed.
  • The colour of any info pane.
  • Which (if any) popup information windows are displayed.
