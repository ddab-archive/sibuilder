Delphi COM Server Applications

SIBuilder supports registration and unregistration of Delphi out-of-process COM server applications, or any other application that uses the /regserver and /uregserver command line switches to perform automatic registration without running the main application.

To create entries for any Delphi out-of-process servers click the Delphi COM Apps button on the Programs page. SIBuilder does not perform any checks on these programs. The button is disabled if there are no 32 bit applications in the project.
