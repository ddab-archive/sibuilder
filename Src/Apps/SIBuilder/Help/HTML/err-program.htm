Program Error

SIBuilder has detected a problem with the program currently being edited. There are two possible reasons for this:

  1. The program file is to be installed from a temporary group (and will therefore be deleted by the installer), but it is flagged for running under the uninstaller.
    Solutions:
    1. change the value of the Execute on drop down list to Installation only;
    2. cancel this dialog box and install the program file into a different non-temporary group using the Files page.
  2. The program is being run from the uninstaller but there is no uninstaller included in the project.
    Solution: cancel this dialog box and use the Project Details page to ensure an uninstall program that is included in the project.
