Edit Projects to be Uninstalled Dialog Box

This dialog box is displayed when the Edit button in the Uninstall project(s) edit box on the Project Details page is clicked.

It is used to define or edit a list of identifiers for applications that are to be uninstalled before the program is installed. The identifiers are those which define the application in the Windows registry's uninstall section.

To edit an existing identifier select it in the All projects to be uninstalled list, click the Edit button, edit it in the Project to be uninstalled and click the Update button to accept the change. To add a new identifier enter it into an empty Project to be uninstalled edit box and click the Add button. Delete an identifier by selecting it in the All projects to be uninstalled list and clicking the Delete button. Change the order in which the programs are uninstalled by selecting an identifier in the list and clicking the and arrow buttons.
