Choose Registry Template Dialog Box

This dialog box appears when the Templates button is clicked on the Registry page. It is used to select one of the pre-defined registry templates and to assign values to the registry "macros" or placeholders within the template.

The keys contained in the chosen template are then added to the registry tree on the registry page. New keys created in this way will be deleted by the uninstaller only if the keys are empty.

All available templates are listed in the Choose template drop down list. When the required template is selected from the list it is displayed in the dialog.

The template can be accepted as-is, or, if it contains registry template macros it can be customised by providing values for the macros. Macros appear as blue hotlinks in the template. A description of the macro will be provided as a tool tip if the mouse cursor is paused over the hotlink.

The macro can be given a new value by clicking the hotlink. A popup menu will be displayed which contains a list of previous values assigned to the macro, the macro name itself and a New Value option. Selecting one of the values or the macro name will cause it to replace the current macro value. Clicking New Value will display the New Registry Macro Value dialog box where a new value can be entered. The template will be updated to display the new value. When the template has the required values click OK to update the registry tree.

The current values associated with any macro can be edited by clicking the Edit Macro Values button and making the required changes in the Edit Registry Template Macros dialog box.
