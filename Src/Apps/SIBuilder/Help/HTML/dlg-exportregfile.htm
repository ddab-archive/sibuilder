Export Registry File Dialog Box

This standard Windows save file dialog box appears when the Export button is clicked on the Registry page.

It is used to name a registry file that is to be exported into the project. The type of export file should be selected in the Save as type drop down list. SIBuilder can export files in RegEdit v4 or v5 format.

If the name of an existing file is selected the user is invited to confirm that the file can be overwritten.
