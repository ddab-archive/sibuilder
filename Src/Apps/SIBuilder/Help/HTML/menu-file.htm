File Menu

The file menu provides the following options:

New Default
(Ctrl+N) 		Creates a new, untitled, project file containing the current default project (if any). If a project is currently in memory it is cleared. You can save any project as the default by using File | Make Default menu option (see below).
New Blank
(Shift+Ctrl+N) 		Creates a new, untitled, blank project. If a project is currently in memory it is cleared.
Open
(Ctrl+O) 		Opens an SIBuilder file from disk. This menu option displays the Open Project standard Windows dialog box. You should choose a .sip or .sib file to open. The newly opened file replaces any current project. If an old style .sib, or a release 2 .sip file is opened then its file name is not stored and the project should be saved under another name. It can be saved only as a release 3 .sip file. You can use the Tools | Convert menu option to convert release 1 .sib files into .sip files before opening them.
Save
(Ctrl+S) 		Writes the current project file to disk. If the current project has not yet been saved then this command acts in the same way as Save As.
Save As
(Ctrl+A) 		Saves the current project under a new name. The Save Project standard Windows save dialog box is displayed. You should provide a file name with extension .sip. This extension is appended by default if no extension is given. The path you specify must exist. You are warned if the file name you specify already exists. SIBuilder 3 cannot save to .sib files.
Make Default Makes the current project the default project.
Clear Default Clears the current default project When this menu option is chosen the File | New Default and File | New Blank menu options have the same effect until a new default project is specified by chosing File | Make Default once more.
Exit This option closes SIBuilder.
