Help Menu

The help menu provides the following options:

Contents Displays the contents dialog box of this help file.
Visit DelphiDabbler.com Goes online and displays the home page of the DelphiDabbler website in the default browser.
Online Tutorial Accesses a tutorial that explains how to use SIBuilder. The tutorial is located on the DelphiDabbler website.
License Displays the SIBuilder license agreement dialog box that shows the license under which SIBuilder is distributed and used.
About Displays an about box giving copyright information about SIBuilder. The about box also provides version information and details of the installed files.
