Save Project Dialog Box

This standard Windows dialog box is displayed when the File | Save As menu option is selected (or File | Save is selected for an un-named project). It can also displayed when an un-saved project is built.

The dialog is used to provide a path and file name for a project file. Files with the .sip extension should be specified. It is not possible to save to SIBuilder v1 .sib format files.

If a chosen file already exists the user will be invited to confirm that the file can be overwritten.
