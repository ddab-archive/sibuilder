Test License Dialog Box

This dialog box shows what the license dialog box that will be displayed by the installer will look like. It is displayed by clicking the Display License button on the License Information page, where its appearance has been configured. The only difference between this dialog box and the one that will be displayed by the installer is that this version has a help button and the one created by the installer does not.

The dialog box is user-configurable using the License Information page. The configurable properties are:

License file If no license file has been specified then a message is displayed to that effect.
Span Small, medium and large sizes can be specified.
Number of buttons Either a single Close button is displayed or a pair of Accept and Decline buttons are displayed. In the first case the installer simply displays the dialog box and continues when it is closed, while in the latter case installation only proceeds if the Accept button is clicked.
