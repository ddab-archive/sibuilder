Overview

SIBuilder Release 3 (Small Installation program Builder) is an open source software installation program creator that creates a stand alone executable with little overhead taken up by the installer itself. This makes the program ideal for creating a single, self-extracting installation program for distribution over the Internet.

To keep down the overhead required for the Installer and the optional uninstall program SIBuilder creates these programs as Windows console applications.

The user specifies the files to be installed which are compressed before being compiled into the installer. Main features are:

  • The installation can be limited to certain operating systems.
  • Different installation folders can be specified for various groups of files. These file groups can be customised for different operating systems.
  • The registry can be updated with the required keys and data – with commonly used registry keys being accessed via templates.
  • The installer can be configured to run further programs to customise installation.
  • In-process COM servers can be registered.
  • A license dialog box can be displayed and the license accepted before the installation proceeds.
  • An optional uninstall program be created and uninstallation triggered from Control Panel's "Add/Remove progams" applet.

Information needed to create an installation project is entered using a multi-page wizard-like interface. Project information can be stored in and read from .sip project files.
