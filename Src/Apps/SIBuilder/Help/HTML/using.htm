Using SIBuilder

SIBuilder installation projects are entered using the program's GUI. The main part of the user interface is the Project Wizard where the project details are entered into the various wizard pages.

You can create new projects or load them and save them, get help and configure SIBuilder using the Main Menu.

Project files can also be opened by:

  • Passing the name of the file on the command line.
  • Dragging and dropping a project file from Explorer.
