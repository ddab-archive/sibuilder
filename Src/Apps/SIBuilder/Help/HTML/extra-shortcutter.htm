Extra Program 2: Shortcutter

This program enables you to create shortcuts (aka shell links) from SIBuilder. The program creates or deletes a shortcut depending on the parameters passed to it.

Shortcutter can be used in SIBuilder's Programs page to create a shortcut as part of an installation. To do this provide parameters as specified below. The [Desktop] and [Programs Menu] path macros can be used to place shortcuts on the desktop or start menu. The program can also be used to delete shortcuts on uninstallation. To use Shortcutter in this way include it in your install project on SIBuilder's Files page.

Creating a shortcut:

To create a shortcut run Shortcutter with the following parameters:

Shortcutter.exe <ShorcutFile> <LinkedFile>
<WorkingDirectory> <Description> <IconFile> <IconIndex>
<Argument1> <Argument2> ... <ArgumentN>
(all on one line)

Description of parameters

<ShortcutFile> Name of the shortcut file to be created. This must have a .lnk extension. This parameter is required.
<LinkedFile> Name of the file to be referenced by the shortcut. This parameter is required.
<WorkingDirectory> Working directory of any program started by shortcut. Optional.
<Description> Description of shortcut. Optional.
<IconFile> File from which shortcut's icon is taken. Optional.
<IconIndex> Index of the required icon in . Must be a valid non-negative number, otherwise zero is assumed. Optional.
<ArgumentX> Zero or more parameters to pass to the program started by the shortcut. Optional. Omit if not required.

Notes

  1. Any parameters containing spaces should be enclosed in double quotes.
  2. An * (asterisk) may be specified for any optional parameter other than <ArgumentX> to indicate that no value is being specified.

Deleting a shortcut

To delete a shortcut use this command line:

Shortcutter.exe <ShorcutFile>

Description of parameters

<ShortcutFile> Name of file to be delected. This parameter is required. There must be no other parameters.

Notes

Note that <ShortcutFile> is checked to see if it is a valid shortcut file and is only deleted if so.
