v2 File Conversion Comment:
License Dialog DLL Removed from Project

SIBuilder uses a DLL – SITLicenseDlg.dll – to display any license required by the installer. SIBuilder 3 deletes the DLL once the license has been displayed, while SIBuilder 2 stores the DLL in the installation folder and only (optionally) deletes it with the uninstaller.

The current case is one where the DLL would have been deleted by the uninstaller under SIBuilder 2. However, it is assumed that the current project does not need the dialog DLL since the license file itself was flagged for deletion by the installer once it had been displayed. Therefore SIBuilder 3 has not taken any steps to retain the DLL after it has been used to display the license.

If it is known that the DLL is required by the project then the DLL should be added to the project either into an existing group or into a newly created one. Use the Files page to make these changes.

SITLicenseDlg.dll forms part of the core SIBuilder installation and may freely be copied and distributed with your application.
