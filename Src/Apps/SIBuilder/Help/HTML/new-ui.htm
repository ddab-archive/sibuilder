Interface Enhancements in SIBuilder 3

Interface enhancements in SIBuilder 3 include:

  • A page selector has been added to the left hand side of the window which is used to select the required pages in the wizard. This replaces the previous sidebar image.
  • New pages have been added to the wizard. The new pages are:
    • An optional Welcome page that replaces the old welcome dialog.
    • A Target OS page that is used to define the operating systems on which the installer will run.
    • A License info page that replaces the previous license dialog box and is used to provide details of any license to be displayed by the installer.
  • In-process COM Servers are now flagged for registration from the Files page.
  • Icons that represent file types have are now display on the Files page, rather than generic icons for all files.
  • The layout of the Programs page has been changed to provide more visual information about programs. Programs are grouped into install and uninstall trees and parameters are all listed under their respective programs.
  • Context sensitive menus have been added to many pages to provide relevant commands depending on the items being edited.
  • New, optional, popup information windows are now available on the Files and Registry pages. These windows provide detailed information about selected objects that was previously only visible if the object's editing dialog was opened.
  • A new about box has been provided that gives information about all the executable files that make up the SITools suite. The dialog also provides access to 3rd party credits and copyright & licensing information.
  • Edit boxes that receive file and folder names now have an integral browse button that opens a suitable dialog box. File related combo boxes now have an entry that accesses dialogs in a similar way.
  • Extended the use of hotlinks to provide quick access to commands and help and to provide additional information.
  • Totally redesigned the Options dialog box.
  • The help file was revised and changed to HTML help as at version 3.4.
  • The Files and Programs pages now have clickable warning hot links that appear when there are problems on the page. Clicking the link displays an indication of what the problem may be.
