Options Dialog Box

This dialog box is displayed by selecting the Tools | Options menu item. It is used configure SIBuilder. The dialog is grouped into two main areas: Build Options and Display Options.

Build Options

Default compression

This combo box lists all available compression systems that can be used to compress files when creating install files – along with the (none) option that enables uncompressed install files to be created. Normally SIBuilder will use the compression system defined in the project file. The selection made here will be used when a new project is created or if the compressor specified by a project is not available on this system. The compressor can be changed on a project by project basis on the Build Installer page.

Report kinds

This combo box lists the formats that are available for reporting on project builds. There are two options: plain text reports or HTML reports. The Report button on the Build Installer page will create reports of the type selected here.

Display Options

Include welcome page

When this item is checked the Welcome page is displayed whenever the program starts up and its button appears in the page selector control. When unchecked the welcome page is not displayed on start-up and its button is not displayed in the page selector.

Show info pane

The info pane is displayed at the top of the program window if this box is checked and is hidden if unchecked.

Info pane colour

This combo box is only enabled if Show info pane is checked. Use the combo box to choose the colour of the info pane. Options are:

Window pane colour Displays the info bar in the current window colour (often white).
Window background colour Displays the info pane in the same colour as a dialog box – it blends into the program's main window.
Custom... Displays the Info Pane Custom Colour dialog box to enable the user to choose any required colour for the info pane.

Display pop-up information windows for

These check boxes enable the user to decide which pop-up information windows to display when certain kinds of items are selected in various pages.
