About Registry Template Macros

Registry template macros are placeholders within registry templates that can be edited to take required values. Each macro remembers the last value assigned to it and that value is used by default in all the registry templates that include the macro.

The previous values that were used in the macro are also remembered so they can easily be restored by choosing them from a popup menu.

The list of values associated with a macro can be edited using the Edit Registry Template Macros dialog box.
