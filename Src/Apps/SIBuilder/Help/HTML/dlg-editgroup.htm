Edit Group Properties Dialog Box

This dialog box is displayed when the Add Group or Edit Group buttons on the Files page are clicked. It is used to either define the properties of a new file group or to edit the properties of an existing group.

The various group properties that can be edited are explained below.

Name of Group

The name of the group is entered in the Group name edit box. This name must be unique.

Installation Path

The path where the files in this group are to be installed are entered in the Installation path combo box. These paths can include path macros, which can be selected from the combo box's drop down list. Alternatively the folder can be chosen from the Choose Folder dialog box by clicking the Browse for folder entry in the combo box. The folder selected is entered in the combo box and can be edited.

Deletion Options

The various options for deleting the group or its files are specified in the Folders and Files radio button groups.

The options for deletion of folder are:

  • Do not remove folder – the folder is never deleted.
  • Remove folder if empty – folder is removed by the uninstaller only if it contains no files or sub folders.
  • Remove folder and all contents – the uninstaller forcibly removes the folder and all its files and sub folders.

Similarly the choices available for deletion of files in the group are:

  • Do not delete files   these files are never deleted.
  • Delete only files that were installed – only files installed by the installer are deleted by the uninstaller - any other files are left intact.
  • Delete all files in folder – all files in the folder, regardless of whether they were installed by the installer, are deleted by the uninstaller.
  • Temporary files deleted by installer – once the files have been used by the installer it deletes them (e.g. use this option for a program and/or file to be run by the installer on installation only).

If you have chosen to remove the folder and its contents the Files radio buttons are disabled and the delete all files option is implied. If any of the Windows, System, Program Files or Common Files folders are specified then it is not permitted to remove the folder or delete all files. If the group's install folder is a special system folder then several deletion operations are not permitted.

For further information about file and folder removal options see the File Groups help topic.

Operating System Options

Each group can be set so that it installs only on certain operating systems. This enables different files to be installed on different operating systems. The selected operating systems for this group are listed in the Operating system options group box. The supported operating systems can be edited using the Operating System Options dialog box which is accessed using the Select Options button.
