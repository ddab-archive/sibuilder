License Information Page

This page is used to define which, if any license is displayed by the installer.

If a license is to be displayed a special DLL required by the installer to display the license is automatically included in the installation. The user must provide the license file to that is to be displayed. The installer deletes both the license and the DLL once the license has been shown.

If the license file is also to be used by the program being installed it must be explicitly added to the project on the Files page. Developers are free to use the license DLL with their programs – again the DLL needs to be explicitly added to the project in such cases. In either case no additional overhead is incurred since each file is included only once in the install program, regardless of the number of copies to be installed.

The license page is divided into four main areas. Each of the key areas is described below.

Display License Check Box

Check the Display license check box if a license is to be displayed by the installer. If this check box is clear then the remainder of the page is disabled and there is no more to do. If the box is checked then, at a mimimum, a license file must be provided.

License File Group Box

Information about the license file to be displayed is entered here. Enter the source license file name and path in the Source license file edit box, or click the button embedded in the edit box to display the Browse for License File standard windows dialog and use that to find the license file.

The license file can be dragged and dropped onto the group box from Explorer.

Dialog Box Properties Group Box

In this group box the characteristics of the license dialog box are defined.

The Which buttons drop-down list allows you to choose the buttons that are displayed as follows:

  1. Accept and Decline: two buttons are displayed and the user is required to accept the license before proceeding with the installation. Should the user decline then the installation is halted.
  2. Close only: a single button is displayed and installation simply continues when the dialog box is closed.

The dialog box can be displayed in one of three sizes – small, medium and large. Choose the required size from the Dialog size drop-down list.

Test Group Box

Click the Display License button to display the Test License dialog box according to the defined properties. If no license file has been selected, or the chosen file doesn't exist a message to this effect is displayed in the dialog box.

The license dialog box displays a Help button to enable you to get further information about the dialog box. This button will not appear when the dialog box is displayed by the installer. The buttons requested in the Which buttons drop down list are displayed but serve only to close the dialog box.
