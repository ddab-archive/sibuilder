Extra Program 1: RegInprocSvr

This program enables the user to register and unregister in-process COM server DLLs via a Windows interface. It is a windowed equivalent of the Windows RegSvr32 command line application.

To run the program simply double-click it in the Extras folder.

Note: If running on Windows Vista an elevation prompt may appear and an administrator password may be required.

Enter the full file name of the DLL to be registered or unregistered (or drag and drop it from explorer) then click either the Register or Unregister buttons as required. If the registration or unregistration is successful then a confirmation message appears at the bottom of the window, otherwise an error message is displayed.

Note: Should the DLL fail to register on Windows NT, 2000 or XP you should check you have sufficient privileges to register a COM server. If not, log on as an administrator.

The program requires RegCOMSvrs.dll, itself a COM server. This file is located in the same folder as the program and is registered when SIBuilder is installed.
