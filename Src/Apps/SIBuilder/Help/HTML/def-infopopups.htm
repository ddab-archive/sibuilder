About Popup Information Windows

Pop-up information windows can appear when items in some pages are selected. These information windows provide further information about the selected item. Clicking the window's close button or clicking outside the window causes it to be closed. The window can be dragged using its title bar.

The following list tells which items have associated information windows and describes the information displayed in the window:

Files page

Group nodes Displays the group name, install path, removal options and supported operating systems.
File nodes Displays the file name, type of file, overwrite options and special options.

Registry page

Registry keys Displays the key name and its deletion options.
Registry data Displays the data item name, data type and (possibly truncated) value.

About box

Displays additional information for a files installed with SITools.

Except for the About Box, the display of pop-up windows can be turned on or off by using the Display popup information windows check list in the Options dialog box.
