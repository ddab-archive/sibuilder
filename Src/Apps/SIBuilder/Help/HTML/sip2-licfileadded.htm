v2 File Conversion Comment:
License File Added to Project

SIBuilder 3 always deletes license files after use. By default, SIBuilder 2 acted in the same way, but the user could specify that the file should be kept after use and be deleted by the uninstaller. The file would be installed in a user-defined folder.

This is the case here, so SIBuilder 3 has created a new group – "SIP2 CONVERTER: License File" which contains the license file and installs it to the appropriate folder, flagging it for deletion by the uninstaller.

In this case the DLL used to display the license will also have been added to the project in the "SIP2 CONVERTER: License DLL" group.

If the license and/or the DLL are not required in the project, they can be deleted from the Files page by selecting the group(s) and clicking the Delete Group button.
