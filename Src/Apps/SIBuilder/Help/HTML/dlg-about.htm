About Dialog Box

The about box displays information about the SITools suite. It is accesssed from the Help | About menu option or a hotlink on the Welcome page.

The top part of the dialog box describes the programs and contains various hotlinks – to the end user license agreement and credits help topics and to the DelphiDabbler website.

The various component files that make up the SITools suite are listed at the bottom of the dialog. The version number and a brief description are displayed. Clicking any file name that appears as a hotlink causes a pop-up information window containing additional information about the file to be displayed. Simply click the pop-up window's close button or click outside it to close the pop-up.
