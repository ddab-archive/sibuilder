Credits

SIBuilder was created using Borland Delphi.

Several of the author's publicly available open source components were used in the construction of the program. These are available from the author's website.

Two different third party sources were used to create the inflation and deflation DLLs used to compress and expand installation projects. They are:

  • The BZip2 library: BZip2 v0.9.5d code by Julian Seward. Delphi wrapper code was adapted by the author from BZip2 unit v1.01 by Edison Mera Menéndez.
  • The ZLib library: the Delphi wrapper for the ZLib code was based on a unit provided with Borland Delphi. The ZLib library is copyright © 1995-1998 Jean-loup Gailly and Mark Adler.
