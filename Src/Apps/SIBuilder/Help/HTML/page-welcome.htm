Welcome Page

When SIBuilder starts up the first time it displays the Welcome page. The page gives a brief overview of SIBuilder and provides a means of jumping off to do some useful start-up tasks. These tasks are accessed from the bulleted list by selecting the blue highlighted links with the mouse.

Further links lead to copyright information and the program's about box.

This page can be switched off when no longer needed by deselecting the Include welcome page checkbox in the Options dialog box.
